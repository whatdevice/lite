About WhatDevice

WhatDevice is a web app designed to show information about your computer, tablet, phone, or other internet-connected device. It does all this through the magic of JavaScript - you don't have to download any programs or apps!

You're using WhatDevice Lite, a custom version of WhatDevice designed to run on legacy browsers (like Internet Explorer 8 and below) or basic WAP browsers (like Opera Mini). If you're using a recent browser, please use the full version instead.

The ability to share a report is coming soon!
